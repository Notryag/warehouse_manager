控制springboot版本有2中方式

方式一：
    在创建的boot工程中pom中使用parent继承springboot的父工程
    同时指定版本 背后其实也是指定springboot的夫工程继承的springboot
    版本仲裁中心工程的版本，依赖来控制springboot所有相关依赖的版本

方式二：
    在依赖版本控制标签dependencymanagement 中引入Springboot版本仲裁中心，
